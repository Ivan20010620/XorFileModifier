# XorFileModifier

Десктопное приложение для побайтового XOR-шифрования файлов с графическим интерфейсом на Qt 6.

## Возможности

- Пакетная обработка файлов по маске (например `*.txt`, `*.bin`)
- XOR-ключ до 8 байт (задаётся в hex-формате)
- Выбор каталогов ввода и вывода
- Режим перезаписи или автоматическое переименование дубликатов
- Опция удаления исходных файлов после обработки
- Таймер для периодического опроса каталога и автоматической обработки новых файлов
- Отмена обработки в любой момент
- Прогресс-бар и лог операций

## Требования

- C++17
- Qt 6 (модуль Widgets)
- CMake 3.16+

## Сборка

```bash
mkdir build && cd build
cmake ..
cmake --build .
```

## Использование

1. Укажите каталог с входными файлами
2. Укажите каталог для сохранения результатов
3. Задайте маску файлов (например `*.dat`)
4. Введите XOR-ключ в hex-формате (до 16 символов / 8 байт)
5. Нажмите **Запустить** для однократной обработки или **Запустить таймер** для периодической
